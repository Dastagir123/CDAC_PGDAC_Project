# CDAC_PGDAC_Project
CDAC PGDAC Project
Title : Course Management System (CDAC)
Technology : Spring Boot (Java), Thymeleaf, Html &CSS,
 JavaScript. MySQL.
Duration: 2 Months
Description : Aim of this project is to provide platform for teachers to upload course
material and attendance and details about examination. Students will
able to access all the information regarding whole course on student
section with their respective ids and passwords. In admin section of the
webapp admin will have access to all student data an teacher data. Admin
will have permissions to perform CRUD operations on Students,
Teachers and course data.
